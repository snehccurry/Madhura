# Madhura

Madhura is a productivity app that helps users stay focused by requiring them to plan their current hour and periodically checking if they remain on task.


<img width="1920" height="1200" alt="image" src="https://github.com/user-attachments/assets/d438b019-097d-4c06-924f-19c6fa8dc345" />



## Overview

Madhura reduces distractions by introducing structured hourly planning and frequent focus check-ins.

Instead of letting the next hour pass without intention, the app requires the user to define exactly what they will work on. Once a task is set, Madhura periodically checks in to ensure the user remains focused.


## Features

- Hourly planning — Define the task you will work on for the current hour
- Forced task selection — The app stays active until a task is planned
- 7-minute focus check-ins — Regular prompts verify whether the user is still focused
- Focus streak tracking — Encourages consistent attention and discipline
- Minimal interface — Designed to avoid additional distractions
- Notification reminders — Helps redirect attention back to the task

## How It Works

1. The user defines the task for the current hour.
2. The app requires a task before continuing.
3. Every 7 minutes, a focus check-in appears.
4. The user confirms whether they remained focused.
5. Focus streaks are recorded to encourage consistency.

## Use Cases

Madhura can be used by:

- Developers
- Students
- Designers
- Remote workers
- Anyone looking to reduce distractions and improve deep work
